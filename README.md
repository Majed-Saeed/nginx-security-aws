# Nginx Security Hardening (AWS EC2)

## Overview

This project demonstrates securing an Nginx server on AWS EC2.

## Features

- Rate limiting (basic DDoS protection)
- Connection limiting
- Reverse proxy setup
- HTTPS (Let's Encrypt)

## Tech Stack

- AWS EC2 (Ubuntu)
- Nginx
- Node.js

## Files

| File | Description |
|------|-------------|
| `security.conf` | Rate limiting configuration |

## Goal

Hands-on DevOps practice with real server security setup.
